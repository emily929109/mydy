# Vue 3 Component Master Skill

This skill defines the standards for creating high-quality, reusable Vue 3 components within this project.

## Component Standards

- **SFC Structure**: Use `<script setup>`, `<template>`, and `<style scoped>`.
- **Props**: Define clear, typed props using `defineProps`.
- **Emits**: Define events using `defineEmits`.
- **Styling**: Utilize MDB (Material Design for Bootstrap) classes where possible for consistency.
- **Interactivity**: Use `ref` and `reactive` for state management.
- **Documentation**: Include brief comments explaining the component's purpose and props.

## ArticleCard Requirements

- **Title**: A string prop for the article title.
- **Content**: A string prop for a short description or summary.
- **Image**: A string prop for the image URL.
- **Author**: A string prop for the author's name.
- **Date**: A string prop for the publication date.
- **Link**: A string prop for the "Read More" link destination.
- **Interactivity**: Add a hover effect and a "Like" toggle state.
